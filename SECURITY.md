# Security / Seguridad

## 🇪🇸 Español

Si descubres una vulnerabilidad de seguridad en SynapseCode, por favor repórtala enviando un correo a **oscarfemartin@gmail.com** (o abre un issue con etiqueta `security` si no es crítica).

No publiques la vulnerabilidad públicamente hasta que haya sido resuelta.

## 🇬🇧 English

If you discover a security vulnerability in SynapseCode, please report it via email at **oscarfemartin@gmail.com** (or open an issue with the `security` label if it is not critical).

Please do not disclose the vulnerability publicly until it has been resolved.
