Características

Todo lo que necesitas para debates multi-agente

Desde la asignación inteligente de modelos hasta reportes profesionales en PDF y DOCX.

🧠

Debate Multi-Agente

Orquesta 4-12+ agentes con roles especializados (Analista, Crítico, Sintetizador, Validador) en debates secuenciales o iterativos.

⚖️

Tribunal de Magistrados

Defensor, Fiscal y Árbitro evalúan argumentos con fallback chains automáticos y protocolo de consenso configurable.

🔍

Búsqueda Web en Tiempo Real

DuckDuckGo + Trafilatura inyectan contexto actualizado en los debates para verificación factual automática.

🎯

Asignación Inteligente

Model Registry + Evaluator + Role Matcher asignan el mejor modelo por rol según especialidad, VRAM y rankings en vivo.

📊

Reportes Profesionales

Exporta debates como JSON, Markdown, PDF imprimible, DOCX (Word) o TXT con métricas y veredicto estructurado.

️

Arquitectura Master-Worker

PC Master orquesta, PC Worker ejecuta modelos locales (Ollama, LM Studio, Jan) con auto-recuperación y monitoring GPU.

Inicio rápido

Funcionando en 3 minutos

Requiere Python 3.12+ y acceso a un PC Worker con Ollama (o usa solo modelos cloud).

# 1. Clonar el repositorio git clone https://github.com/OscarFeMa/SynapseCode.git cd SynapseCode # 2. Crear entorno virtual e instalar python -m venv venv .\venv\Scripts\pip install -r backend/requirements.txt # 3. Configurar variables de entorno copy .env.example .env # Editar .env: añadir API keys de Groq/Gemini si se usan # 4. Iniciar el backend start_backend.bat # o manualmente: uvicorn backend.main:app --host 0.0.0.0 --port 8000 # 5. Abrir el panel de administración # http://localhost:8000/admin
