El Motor de Debate
Multi-Agente Definitivo.

Plataforma de razonamiento colectivo híbrido que orquesta múltiples modelos de IA en debates estructurados con un Tribunal de Magistrados autónomo.

Explorar en GitHub

Debate Estructurado

Orquesta roles específicos (Analista, Crítico, Sintetizador) para explorar todas las facetas de un problema complejo y evitar el sesgo cognitivo.

Tribunal Autónomo

Los resultados son evaluados por Magistrados (Defensor, Fiscal, Árbitro) garantizando la veracidad, el consenso y descartando falacias.

Búsqueda Web en Vivo

Los modelos no solo dependen de su entrenamiento. El motor incorpora resultados web frescos para discutir sobre información de actualidad.

Reportes Profesionales

Cada debate se sintetiza automáticamente en exportaciones listas para usar: HTML interactivo, PDF elegante, DOCX estructurado y JSON.

Arquitectura Híbrida

SynapseCode integra una red P2P (Master-Worker) donde los agentes pesados corren en la nube (Groq, Gemini, OpenRouter) y los agentes de privacidad o baja latencia corren localmente (Ollama, LM Studio).

Arranca el Backend .\venv\Scripts\python -m uvicorn backend.main:app --host 0.0.0.0 --port 8000
